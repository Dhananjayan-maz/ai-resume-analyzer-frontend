# 🚀 AI Resume Analyzer - Frontend

A modern and responsive React application that allows users to upload resumes, receive AI-powered resume feedback, and view their resume analysis history.

## 🌐 Live Demo

**Frontend:** https://ai-resume-analyzer-frontend-vcpi.onrender.com

**Backend API:** https://ai-resume-analyzer-backend-dii1.onrender.com

---

## 📌 Features

* User Registration
* Secure JWT Authentication
* Login & Logout
* Upload Resume (PDF)
* AI-powered Resume Analysis
* Predicted Job Role Display
* Resume History
* Responsive User Interface
* Toast Notifications
* Protected Routes

---

## 🛠️ Tech Stack

* React 19
* Vite
* React Router DOM
* Axios
* Bootstrap 5
* Bootstrap Icons
* React Toastify
* AOS (Animate On Scroll)

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Dhananjayan-maz/ai-resume-analyzer-frontend
```

Go to the project

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build for production

```bash
npm run build
```
